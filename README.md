# Analysis of Fake Currency Circulation in India & Detection of Fake Currency Using Statistical Image Processing and Machine Learning

Counterfeit currency poses a serious challenge to Indiaâ€™s economic stability and public trust, contributing to financial losses and supporting illegal activities. As fake notes become more sophisticated, traditional detection methods are increasingly inefficient. This project analyzes counterfeit currency trends across Indian states using official government and law-enforcement data to identify high-risk regions. It also proposes a cost-effective, automated fake note detection system using image processing and machine learning. Trained on the Banknote Authentication Dataset, the model uses key statistical features to accurately classify genuine and counterfeit notes, offering a scalable solution for banking, retail, and digital applications.
